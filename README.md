# TadHack 2020 Stuff

## Setup

Configure `.enf` file as follows:

```.env
# Misc
NGROK_URL='https://76d473358193.ngrok.io/webhook'

RECIPIENT_PHONE='E164 format of phone number to be used to receive calls'

# Ring Central
RINGCENTRAL_CLIENT_ID=''
RINGCENTRAL_CLIENT_SECRET=''
RINGCENTRAL_SERVER='https://platform.devtest.ringcentral.com'
RINGCENTRAL_USERNAME='E164 format username from RingCentral account'
RINGCENTRAL_PASSWORD=''
RINGCENTRAL_EXTENSION='101'

# Avaya
AVAYA_USERNAME='E164 format Avaya phone number
AVAYA_SID=''
AVAYA_AUTH_TOKEN=''

# Symbl
SYMBL_APP_ID=''
SYMBL_APP_SECRET=''

```
